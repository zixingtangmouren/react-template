# react-template

一个由 webpack5 从 0 到 1 搭建的 React 项目。

## 功能清单

- [x] 支持 JS 编译，语法兼容
- [x] 支持 TypeScript
- [x] 支持 React JSX 语法
- [x] 支持开发环境调试
- [x] 支持语法规范、代码风格检测
- [ ] 支持 commit 信息规范检测
- [ ] 支持 git 提交代码与检测
- [ ] 支持 css/less/sass 编译
- [ ] 支持样式兼容
- [ ] 支持移动端 rem 适配方案
- [ ] 支持移动端 vw 适配方案
- [ ] 支持图片、字体资源等静态资源打包
- [ ] 支持生产环境打包
- [ ] 开发环境提效
- [ ] 生产环境打包优化和提效
- [ ] 支持 SSR 产物编译
- [ ] 支持 SSR 开发调试
